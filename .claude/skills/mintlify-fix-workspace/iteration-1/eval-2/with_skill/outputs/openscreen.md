29	## 项目简介
30	
31	openscreen 是一个专注于屏幕截图和屏幕录制的轻量级 Python 工具库。该项目旨在提供简单而强大的 API，使开发者能够仅用几行代码即可实现屏幕截图和屏幕录制功能。
32	
33	**核心价值定位：**
34	
35	- **极简主义**：通过简洁的 API 设计降低使用门槛
36	- **快速集成**：支持 pip 一键安装，可快速集成到任何 Python 项目中
37	- **功能专一**：专注于屏幕捕获这一单一功能，不追求大而全
38	
39	**典型使用场景：**
40	
41	```python
42	# 场景1：快速截图
43	import openscreen
44	screen = openscreen.Screen()
45	screen.screenshot()
46	
47	# 场景2：带文件名保存的截图
48	screen.screenshot(filename='my_screenshot.png')
49	
50	# 场景3：屏幕录制
51	recorder = openscreen.Recorder()
52	recorder.record()  # 开始录制
53	recorder.stop()    # 停止并保存
54	
55	# 场景4：指定输出文件名
56	recorder.stop(filename='recording.mp4')
57	```
