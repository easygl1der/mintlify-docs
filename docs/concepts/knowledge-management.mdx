# 个人知识管理

个人知识管理 (Personal Knowledge Management, PKM) 是指个人有意识地组织和管理自己的知识资产，提高生产力和学习效率。

## 为什么需要个人知识管理

现代人每天接触大量信息：

- 阅读文章、文档、书籍
- 观看视频、教程
- 参加会议、讨论

如果没有有效的管理机制，这些信息很快就会被遗忘或难以找到。

## 核心组件

### 1. 收集 (Capture)

将外部信息转化为个人知识资产：

- 使用笔记应用快速记录
- 保存有价值的网页内容
- 截图和标注重要信息

### 2. 组织 (Organize)

建立清晰的结构：

```
知识库/
├── 概念/      # 基础理论和概念
├── 实践/      # 具体操作指南
├── 参考/      # 快速查阅的资料
└── 归档/      # 不再活跃使用的内容
```

### 3. 反思 (Reflect)

定期回顾和深化理解：

- 每周回顾新记录的内容
- 将相关知识建立连接
- 提炼核心要点

### 4. 输出 (Express)

知识只有被使用才能产生价值：

- 写作和分享
- 应用于实际工作
- 教授给他人

## 工具推荐

| 工具类型 | 推荐选项 |
|----------|----------|
| 笔记应用 | Obsidian, Notion, Logseq |
| 文档托管 | Mintlify, GitBook, Docusaurus |
| 任务管理 | Todoist, Things, OmniFocus |

## 最佳实践

1. **保持一致性** - 建立习惯，定期整理
2. **从小开始** - 不要试图一次性整理所有内容
3. **注重链接** - 让相关知识相互关联
4. **定期清理** - 删除过时或无价值的内容
