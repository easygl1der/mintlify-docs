# 个人知识管理

个人知识管理（PKM）是一套帮助你收集、组织和利用知识的系统。

## 方法论

### PARA 方法

- **P**rojects（项目）：当前正在进行的项目
- **A**reas（领域）：长期关注的领域
- **R**esources（资源）：感兴趣但暂不使用的资源
- **A**rchives（归档）：不再活跃但可能有用的内容

### 渐进式写作

先写草稿，逐步完善。不追求完美，先有再优。

## 工具选择

## 最佳实践

1. **每天记录**：养成记录的习惯
2. **定期回顾**：周/月回顾已积累内容
3. **提取精华**：将长文浓缩为关键要点
