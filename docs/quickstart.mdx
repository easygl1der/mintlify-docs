# 快速上手

本指南将帮助你快速了解如何使用这个知识库。

## 基本概念

这个知识库基于 **MDX** (Markdown + JSX) 构建，支持在 Markdown 中嵌入 React 组件。它托管在 Mintlify 上，提供美观的文档阅读体验。

## 添加新笔记

### 步骤 1：确定位置

根据笔记内容选择合适的目录：

- `docs/concepts/` - 概念性内容
- `docs/how-to/` - 实践指南
- `docs/snippets/` - 代码片段

### 步骤 2：创建文件

在对应目录下创建 `.mdx` 文件，并添加 frontmatter：

```mdx
---
title: 你的标题
description: 简短描述
---

# 你的标题

内容在这里...
```

### 步骤 3：更新导航

如果需要在新页面出现在导航中，编辑根目录的 `docs.json` 文件。

## 搜索

使用页面顶部的搜索框可以快速找到你需要的内容。支持中文和英文关键词。

## 快捷命令

以下是一些常用操作的命令示例：

```bash
# 启动本地预览
npm run dev

# 构建生产版本
npm run build

# 部署到 Mintlify
mintlify deploy
```
