# 快速上手

本指南将帮助你在 5 分钟内学会如何使用这个知识库。

## 第一步：浏览文档

使用左侧导航栏浏览不同分类的文档内容。

## 第二步：搜索内容

点击右上角的搜索框，输入关键词快速找到你需要的内容。

## 第三步：Ask AI 助手

遇到问题？点击 **Ask AI** 按钮，AI 助手会基于知识库内容为你解答。

## 第四步：贡献内容

如果你想添加新笔记：

1. 在对应目录下创建新的 `.mdx` 文件
2. 添加正确的 frontmatter
3. 提交到 GitHub 仓库

```bash
git add .
git commit -m "docs: 添加新笔记"
git push
```

部署会自动触发，几分钟后新内容就会上线。
