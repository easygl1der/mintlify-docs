# MDX 示例

MDX 是 Markdown + JSX，可以在 Markdown 中使用 React 组件。以下是常用语法示例。

## Frontmatter

每个 MDX 文件顶部需要包含 frontmatter：

```yaml
---
title: 页面标题
description: 页面描述
---
```

## 标题

```md
# 一级标题
## 二级标题
### 三级标题
```

## 列表

**无序列表**

```md
- 项目一
- 项目二
  - 嵌套
  - 嵌套
- 项目三
```

**有序列表**

```md
1. 第一步
2. 第二步
3. 第三步
```

## Callout 组件

Mintlify 支持多种 Callout 类型：

```mdx
<Note>这是一条普通提示</Note>

<Warning>这是警告信息</Warning>

<Danger>这是危险警告</Danger>

<Tip>这是技巧提示</Tip>
```

### 效果示例

**Note**: 用于补充信息
**Warning**: 需要注意的事项
**Danger**: 危险或错误操作
**Tip**: 有用的小技巧

## Mintlify 特定功能

### 卡片的组

```mdx
<CardGroup>
  <Card title="卡片标题" icon="📚">
    卡片内容
  </Card>
  <Card title="另一张卡片" icon="🔧">
    更多内容
  </Card>
</CardGroup>
```

### Tabs

```mdx
<Tabs>
  <Tab title="Node.js">
    Node.js 相关内容
  </Tab>
  <Tab title="Python">
    Python 相关内容
  </Tab>
</Tabs>
```
