# MDX 示例

MDX 是 Markdown + JSX，可以在 Markdown 中使用 React 组件。

## 基本语法

### 强调

- **粗体**：`**粗体**`
- *斜体*：`*斜体*`
- ~~删除线~~：`~~删除线~~`

### 代码

行内代码：`const x = 1`

代码块：

```javascript
function greet(name) {
  return `Hello, ${name}!`;
}
```

### 引用

> 这是一段引用文本
>
> 可以有多行

## 组件示例

### Callout

```mdx
:::note
这是一条笔记
:::
```

效果：
:::note
这是一条笔记
:::

### 表格

| 功能 | 语法 |
|------|------|
| 粗体 | `**text**` |
| 斜体 | `*text*` |
| 代码 | `` `code` `` |
