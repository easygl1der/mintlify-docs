# 常用命令

日常开发中常用的命令行工具。

## Git

```bash
# 初始化仓库
git init

# 添加文件
git add .

# 提交
git commit -m "message"

# 推送
git push origin main

# 拉取更新
git pull origin main
```

## Node.js

```bash
# 安装全局包
npm i -g <package>

# 安装项目依赖
npm install

# 运行脚本
npm run dev
```

## Mintlify

```bash
# 安装 CLI
npm i -g mint

# 本地预览
mint dev

# 构建生产版本
mint build
```

## 文件操作

```bash
# 创建目录
mkdir -p path/to/directory

# 复制文件
cp source destination

# 移动文件
mv source destination
```
