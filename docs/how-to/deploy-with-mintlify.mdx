# 用 Mintlify 部署文档

Mintlify 是一个现代化的文档网站生成器，支持 MDX，部署简单，界面美观。

## 前提条件

- Node.js 18+
- Git
- GitHub 账户

## 快速部署

### 方式一：使用 Mintlify CLI

```bash
# 安装 Mintlify
npm i -g mintlify

# 进入文档目录
cd your-docs-folder

# 启动预览
mintlify dev

# 构建生产版本
mintlify build

# 部署
mintlify deploy
```

### 方式二：连接 GitHub

1. 访问 [mintlify.com](https://mintlify.com)
2. 使用 GitHub 登录
3. 选择要部署的仓库
4. 自动部署配置

## 目录结构

```
your-docs/
├── docs.json          # 配置文件
├── docs/
│   ├── index.mdx      # 首页
│   ├── quickstart.mdx
│   └── .../
├── .mintlify/
│   └── Assistant.md   # AI 助手配置
└── images/
    ├── logo.svg
    └── favicon.svg
```

## 配置文件

`docs.json` 是核心配置文件：

```json
{
  "name": "你的文档名称",
  "theme": "mint",
  "colors": {
    "primary": "#16A34A",
    "light": "#f0fdf4",
    "dark": "#166534"
  },
  "navigation": [
    {
      "group": "分组名称",
      "pages": ["page1", "folder/page2"]
    }
  ]
}
```

## 自定义样式

### 修改 Logo

将 `images/logo.svg` 替换为你的 Logo。

### 修改颜色

编辑 `docs.json` 中的 `colors` 配置：

```json
"colors": {
  "primary": "#your-color",
  "light": "#your-light-color",
  "dark": "#your-dark-color"
}
```

## 添加新页面

1. 在 `docs/` 下创建 `.mdx` 文件
2. 添加 frontmatter：

```mdx
---
title: 页面标题
description: 页面描述
---

# 内容
```

3. 更新 `docs.json` 中的 `navigation` 配置

## 预览和调试

```bash
# 启动本地预览服务器
mintlify dev

# 查看预览效果
# 访问 http://localhost:3000
```

## 常见问题

### 部署后页面没有更新？

检查 GitHub Actions 是否运行成功，或者手动触发重新部署。

### 自定义域名？

在 Mintlify Dashboard 的 Settings 中配置自定义域名。
