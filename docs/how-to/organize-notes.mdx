# 整理笔记的方法

让笔记井然有序的实用技巧。

## 按主题分类

```
docs/
├── concepts/      # 概念性知识
├── how-to/        # 操作指南
├── snippets/      # 代码片段
└── reference/     # 参考资料
```

## 命名规范

- 文件名使用英文小写和连字符
- 示例：`knowledge-management.mdx`

## Frontmatter 优化

```mdx
---
title: "页面标题"
description: "简短描述（用于 SEO 和预览）"
sidebarTitle: "导航显示的名称"
---
```

## 定期整理

建议每周花 15 分钟：
1. 归档不再活跃的笔记
2. 合并重复内容
3. 更新过时信息

## 链接策略

在相关页面之间建立链接：

```mdx
想了解更多，请参见[知识管理概念](/concepts/knowledge-management)。
```
