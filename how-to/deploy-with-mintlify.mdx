# 用 Mintlify 部署文档

使用 Mintlify 将你的笔记变成专业文档网站。

## 前提条件

- GitHub 账号
- Node.js 18+

## 部署步骤

### 1. 创建仓库

在 GitHub 创建新仓库，或者使用现有仓库。

### 2. 安装 Mintlify CLI

```bash
npm i -g mint
```

### 3. 本地预览

```bash
mint dev
```

访问 http://localhost:3000 预览。

### 4. 推送到 GitHub

```bash
git init
git add .
git commit -m "feat: initial documentation"
git branch -M main
git remote add origin https://github.com/username/repo.git
git push -u origin main
```

### 5. 连接 Mintlify

1. 访问 [mintlify.com](https://mintlify.com)
2. 连接你的 GitHub 仓库
3. 选择要部署的分支

部署完成后，你会获得一个 `your-docs.mintlify.app` 的域名。

## 启用 AI 助手

1. 升级到 Pro 套餐
2. 在仓库根目录创建 `.mintlify/Assistant.md`
3. Mintlify 会自动启用侧边 AI 问答

## 持续更新

每次 push 到 GitHub，Mintlify 会自动重新部署。

```bash
git add .
git commit -m "docs: 更新内容"
git push
```

几分钟后，更新就会生效。
