## 项目简介

TrendRadar 是一个用于获取和展示 GitHub 趋势项目的命令行工具。该工具通过调用 GitHub API 获取指定时间范围内的热门仓库数据，并以格式化的表格形式在终端展示结果。项目采用轻量化的设计理念，代码结构简洁清晰，适合作为学习 Python CLI 开发的参考项目。

## 代码结构

### 3.1 目录结构

```
TrendRadar/
├── src/
│   ├── __init__.py
│   ├── config.py          # 配置文件 (~40-50 行)
│   ├── formatting.py      # 格式化输出 (~30-40 行)
│   └── github_fetcher.py  # GitHub API 封装 (~80-100 行)
├── scripts/
│   └── git_trending.py    # 主入口脚本 (~150-200 行)
└── README.md
```

### 3.2 关键代码文件分析

**src/config.py** - 配置管理模块
- 管理日期范围选项（daily/weekly/monthly）
- 存储 API 相关配置参数
- 提供配置常量供其他模块调用

**src/github_fetcher.py** - 数据获取模块
- 实现 GitHub API 调用封装
- 处理仓库数据解析
- 支持时间范围筛选

**src/formatting.py** - 输出格式化模块
- 利用 prettytable 库渲染表格
- 定义终端输出的列格式和样式

**scripts/git_trending.py** - 主入口脚本
- 集成 argparse 实现命令行参数解析
- 协调各模块完成完整的数据获取和展示流程
