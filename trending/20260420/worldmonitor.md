---

## 项目简介

WorldMonitor 是一款轻量级的全球信息监控桌面应用，由作者 @koala73 开发并托管在 GitHub 平台。该应用的核心价值在于将分散在不同来源的全球信息进行聚合展示，为用户提供一站式的城市信息监控体验。

### 核心功能

根据 README.md 和项目描述，该应用提供以下四项主要功能：

1. **实时天气更新（Real-time weather updates）**
   - 从 OpenWeatherMap API 获取全球城市的天气数据
   - 展示温度、湿度、天气状况等基本信息

2. **最新新闻头条（Latest news headlines）**
   - 集成 NewsAPI 获取全球新闻
   - 按城市或地区展示相关新闻条目

3. **汇率追踪（Exchange rate tracking）**
   - 支持多种货币的汇率显示
   - 便于用户了解各国货币对比

4. **世界时钟（World clock for multiple cities）**
   - 同时显示多个城市的本地时间
   - 支持添加城市到个人监控列表

### 目标用户

- 需要监控多城市天气信息的商务人士
- 关注国际新闻和汇率的个人用户
- 学习 Java Swing 桌面应用开发的开发者
