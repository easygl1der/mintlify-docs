## 项目简介

**wifi-densepose** 是一个极具创新性的开源项目，中文名为"WiFi DensePose"或"π RuView"。该项目旨在将普通的商用 WiFi 信号转换为实时的人体姿态估计、生命体征监测和存在检测能力，整个过程**无需使用任何视频像素数据**。

该项目的核心价值在于利用 WiFi CSI（Channel State Information，信道状态信息）数据进行人体感知。CSI 是 802.11n/ac 协议中定义的物理层信息，记录了无线信号在传播过程中的幅度和相位变化。通过分析这些信号特征，结合深度学习技术，系统可以推断出人体位置、姿态甚至生命体征。

**核心功能特性**：

1. **人体姿态估计**：基于 WiFi 信号实时检测人体的骨骼关键点（如头部、肩部、手臂、腿部等），输出 17 或 25 个身体关键点坐标。

2. **生命体征监测**：通过分析呼吸和心跳引起的胸腔微动对 WiFi 信号的调制，实现非接触式的呼吸频率和心率估计。

3. **存在检测**：判断指定区域内是否有人存在，可用于智能家居的自动控制、安防报警等场景。

4. **多目标跟踪**：支持同时检测和跟踪多个人的运动轨迹。

**典型应用场景**：

| 应用领域 | 具体场景 |
|----------|----------|
| 智能家居 | 人在灯亮、空调自动调节、智能音箱唤醒 |
| 医疗健康 | 老年人跌倒检测、睡眠监测、长期照护 |
| 安防监控 | 入侵检测、周界警戒、异常行为识别 |
| 老年护理 | 独居老人活动监测、紧急情况报警 |
| 工业自动化 | 工厂人员管理、安全区域监控 |

## 技术栈分析

### 编程语言架构

wifi-densepose 采用多语言混合开发策略，根据不同层级的性能和功能需求选择最适合的编程语言：

**1. Rust（主语言，48.3%）**

Rust 是本项目的核心开发语言，承担信号处理、模型推理、业务逻辑等关键任务。选择 Rust 的主要理由包括：

- **内存安全**：Rust 的所有权系统和借用检查器可以在编译期消除内存安全问题，减少系统崩溃和数据泄漏风险。
- **高性能**：零成本抽象和无垃圾回收机制使其适合对延迟敏感的实时处理场景。
- **并发安全**：类型系统确保多线程并发操作的安全性。
- **跨平台**：支持从服务器到嵌入式设备的全平台部署。

Rust 模块分布在 `src/` 和 `rust/` 目录中，主要包括信号处理、模型推理、姿态估计等功能。

**2. Python（35.2%）**

Python 用于机器学习模型的训练、数据预处理和可视化分析。Python 生态中有丰富的深度学习框架和科学计算工具，能够显著提升算法开发和实验效率。

Python 模块位于 `python/` 目录，包含模型定义、训练脚本、数据处理工具等。

**3. C/C++（12.5%）**

C/C++ 主要用于 ESP32 嵌入式固件开发，代码位于 `firmware/` 目录。由于 ESP32 采用 Xtensa 架构，资源受限且需要直接操作硬件外设，C/C++ 是最佳选择。

**4. 其他（4.0%）**

包括 Shell 脚本（构建和部署脚本）、配置文件（TOML、YAML）和文档（Markdown）。

## 技术亮点

### 1. 创新性技术 ⭐⭐⭐⭐⭐

#### CSI 信号人体感知技术

这是本项目最核心的创新点。项目利用 802.11n/ac 协议中的 CSI（Channel State Information）数据进行人体感知：

```
技术原理：
1. WiFi 信号在室内传播时会因墙壁、家具、人体等物体产生反射和散射
2. 人体运动会引起信号多径效应的变化
3. CSI 记录了每个子载波的幅度和相位信息
4. 通过分析 CSI 的时序变化，可以推断人体的运动和姿态
```

**技术优势**：

- 无需部署专用传感器，利用现有的 WiFi 基础设施
- 可以穿透墙壁（有一定限制），实现非视距感知
- 成本低廉，只需普通 WiFi 设备

#### 无摄像头的隐私保护方案

项目实现了真正的"无摄像头"人体感知：

```
隐私保护特性：
1. 不采集任何图像或视频数据
2. 不涉及人脸识别或生物特征识别
3. 仅处理无线信号统计特征
4. 可在本地设备完成全部处理，无需上传云端
```

这使得项目特别适合对隐私要求严格的场景，如卧室、浴室的智能家居控制，或者医疗健康监测。

#### 多模态感知融合

单一系统实现多种感知能力的融合：

| 感知模式 | 技术实现 | 输出 |
|----------|----------|------|
| 姿态估计 | 深度神经网络 | 17/25 个人体关键点坐标 |
| 生命体征 | 信号频域分析 | 呼吸频率、心率估计 |
| 存在检测 | 信号能量分析 | 人员计数、位置定位 |
