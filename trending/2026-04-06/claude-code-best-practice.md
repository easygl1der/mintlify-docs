## 项目简介

### 项目定位

**shanraisshan/claude-code-best-practice** 是一个专注于 Claude Code（Anthropic 开发的 AI 编程助手）的最佳实践指南仓库。该项目旨在为开发者提供系统性的配置指南、使用技巧和工作流程建议，帮助用户充分发挥 Claude Code 的潜力。

### 核心目标

1. **配置优化**：整理 Claude Code 的最佳配置方案
2. **效率提升**：提供实用的使用技巧和快捷方法
3. **质量保障**：指导如何利用 Claude Code 进行测试、代码改进和重构
4. **知识沉淀**：建立可持续更新的最佳实践知识库

### 目标用户群体

- Claude Code 新用户（入门指南）
- 中级用户（效率提升技巧）
- 高级用户（高级配置和工作流优化）
- 技术团队（团队协作规范制定参考）
