## 项目简介

**claude-howto** 是一个精心策划的 Claude 使用指南和资源集合仓库，由开发者 @luongnv89 创建并维护。该项目旨在帮助用户（无论是开发者、内容创作者还是商业专业人士）充分利用 Anthropic 开发的人工智能助手 Claude。

### 项目目标

- 📚 **Comprehensive Documentation（全面文档）**：涵盖使用 Claude 各个方面的组织良好的资源
- 🎯 **Practical Prompts（实用提示）**：针对常见任务和场景的即用型提示模板
- 💼 **Business Applications（商业应用）**：集成 Claude 到组织业务流程的专用指南
- 👨‍💻 **Development Guides（开发指南）**：使用 Claude Code 和 Claude Code MCP 的技巧和最佳实践
- 🖼️ **Visual Aids（视觉辅助）**：演示 Claude 功能的截图和图片

### 内容结构说明

| 目录 | 用途 |
|------|------|
| claude-code/ | 专门针对 Claude Code CLI 工具的指南和提示 |
| claude-code-mcp/ | 专注于 MCP（Model Context Protocol）与 Claude Code 的集成配置 |
| claude-for-business/ | 针对在组织环境中实施 Claude 的商业特定用例和策略 |
| claude-for-development/ | 开发相关资源，包括代码示例、调试技巧和最佳实践 |
| files/ | 附加文档、参考资料和补充材料 |
