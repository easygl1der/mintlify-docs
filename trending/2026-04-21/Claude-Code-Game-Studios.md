## 项目简介

Claude-Code-Game-Studios 是一个专注于展示如何使用 Claude Code（Anthropic 官方 CLI 工具）开发游戏的项目集合。该仓库由 @Donchitos 创建和维护，旨在为开发者提供真实可运行的 AI 辅助游戏开发案例。

**项目定位**

- **教育导向**：通过实际游戏项目展示 AI 编程助手的应用实践
- **代码参考**：提供可学习的游戏开发模板和最佳实践
- **社区共享**：鼓励开发者基于 Claude Code 创建自己的游戏作品

## 可运行性评估

### 运行前提条件

| 条件 | 说明 |
|------|------|
| Claude Code 安装 | 必须安装 Anthropic Claude Code CLI |
| API 密钥配置 | 需要配置 ANTHROPIC_API_KEY 环境变量 |
| 语言运行时 | Python (如涉及 Python 游戏) / Node.js (如涉及 JS 游戏) |
| 特定游戏依赖 | 各游戏项目可能有额外依赖 |

### 快速启动指南

```bash
# 1. 安装 Claude Code
npm install -g @anthropic-ai/claude-code

# 2. 配置 API 密钥
export ANTHROPIC_API_KEY="your-api-key"

# 3. 克隆仓库
git clone https://github.com/Donchitos/Claude-Code-Game-Studios.git

# 4. 进入项目目录
cd Claude-Code-Game-Studios

# 5. 按需运行具体游戏项目
```

**可运行性评级**：⭐⭐⭐⭐⭐
