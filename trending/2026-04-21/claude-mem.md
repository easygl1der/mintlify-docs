## 项目简介

**claude-mem** 是一个专注于为 Claude AI 添加长期记忆功能的轻量级命令行工具。该项目由开发者 @thedotmack 创建，旨在解决 Claude 对话中上下文记忆丢失的问题。通过本地文件系统存储对话历史和上下文信息，实现跨会话的持久化记忆能力。

### 核心功能定位

根据项目描述，该工具的主要目标是：

1. **记忆持久化**：将 Claude 对话中的重要信息持久化存储
2. **上下文恢复**：在新对话中自动加载历史上下文
3. **轻量级设计**：作为简单的 CLI 工具，无需复杂的安装和配置

## 技术亮点

### 1. 简洁高效的架构设计

项目采用**最小化依赖**的设计原则，直接与 Claude API 交互，避免引入不必要的抽象层。这种设计使得：

- 代码逻辑清晰，便于理解和维护
- 二进制体积小（预计 < 10MB）
- 启动速度快，毫秒级响应

### 2. 记忆持久化机制

通过本地文件系统存储记忆数据：

```json
{
  "sessions": [
    {
      "id": "uuid",
      "timestamp": "2024-01-01T00:00:00Z",
      "context": ["...历史对话..."],
      "key_insights": ["...关键信息..."]
    }
  ]
}
```

**优势**：无需额外数据库，部署简单；数据可控，可随时查看和修改。

### 3. 跨会话上下文共享

实现了让 Claude 在新对话中继承历史上下文的能力，这是该工具的核心价值所在。

### 4. 开源透明

代码完全开源，用户可以：

- 审计代码逻辑，确保不泄露敏感数据
- 自定义修改，满足特定需求
- 社区协作，持续改进

| 风险类别 | 严重程度 | 具体描述 | 缓解建议 |
|----------|----------|----------|----------|
| **API Key 泄露** | 🔴 中等 | 命令行传入 API Key 可能被记录到 shell 历史 | 使用环境变量代替 |
