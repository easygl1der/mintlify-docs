## 项目简介

**learn-claude-code** 是一个教育导向的 AI Agent 开发学习项目，其核心描述为："Bash is all you need - A nano claude code–like 「agent harness」, built from 0 to 1"。项目旨在帮助开发者从零开始构建一个类似 Claude Code 的智能体框架（Agent Harness），强调使用 Bash 作为核心实现语言，体现了极简主义的设计理念。

项目定位明确，主要面向以下学习目标：

1. **理解 Agent 原理**：深入理解 AI Agent 的工作原理、架构设计和实现机制
2. **从零构建框架**：手把手教学，从零开始实现完整的 Agent Harness
3. **掌握多语言开发**：学习 TypeScript、Python 与 Bash 的混合开发模式
4. **实践导向学习**：强调实用性，通过实际代码实现完整功能
5. **代码即学习**：提供完整的源代码作为学习素材

项目还配套了独立的学习网站（learn.shareai.run），形成了完整的教学生态系统，这在同类开源项目中较为罕见，体现了作者对教育事业的投入和对项目长期发展的规划。

### 结构设计特点

1. **清晰的分层架构**：源代码按照 agents（智能体）、core（核心功能）、cli（命令行接口）、utils（工具函数）进行分层组织，职责明确。

2. **模块化设计**：每个功能模块独立目录，如 `src/agents/`、`src/core/`、`src/utils/`，便于理解和维护。

3. **脚本分离**：`scripts/` 目录独立存放 Bash 脚本，与 TypeScript 代码分离，体现了"Bash is all you need"的设计理念。

4. **多语言分离**：TypeScript 和 Python 代码分别位于 `src/` 和 `python/` 目录，避免混淆。

5. **文档完善**：独立的 `docs/` 和 `examples/` 目录提供详细的使用指南和示例代码。

6. **测试规范**：单元测试和集成测试分离，测试文件结构清晰。
