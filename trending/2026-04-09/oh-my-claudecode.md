## 项目简介

**oh-my-claudecode** 是一个精心策划的提示词、配置和最佳实践集合项目，旨在最大化 Claude Code 的生产力。该项目类似于 oh-my-zsh 的设计理念，但专门面向 AI 编程助手 Claude Code 用户。

项目的核心价值在于：

1. **提示词库 (Prompt Library)**：提供针对代码审查、调试、重构等多种开发场景优化的提示词集合
2. **Claude Code 配置**：预配置的最佳性能设置，帮助用户快速优化开发环境
3. **安装脚本**：简易的安装和配置脚本，实现一键部署
4. **最佳实践指南**：有效提示词工程的指南和使用示例

### 项目定位

本项目定位为 **Claude Code 生产力提升工具库**，主要服务于以下用户群体：

- Claude Code 新用户快速上手
- 开发者获取优化后的提示词模板
- 团队共享统一的提示词工程实践
- 希望自定义 Claude Code 配置的高级用户

```bash
cat prompts/code_review.md
# 复制内容粘贴到 Claude Code
```

### 安装脚本分析

**install.sh 核心逻辑**:

```bash
#!/bin/bash
# 1. 创建用户配置目录 (XDG 标准)
mkdir -p ~/.config/oh-my-claudecode
mkdir -p ~/.local/share/oh-my-claudecode/prompts

# 2. 复制提示词库
cp -r prompts/* ~/.local/share/oh-my-claudecode/prompts/

# 3. 复制配置文件
cp CLAUDE.md ~/.config/oh-my-claudecode/
cp claude_code_config.md ~/.config/oh-my-claudecode/
```

**脚本质量评估**：
- ✅ 使用 `-p` 参数防止目录已存在错误
- ✅ 使用 `-r` 参数递归复制目录内容
- ✅ 遵循 XDG 目录规范


提示词按照实际开发场景进行组织，而非按照技术类型。这种设计更符合开发者的使用习惯，可以快速找到所需的提示词。
