## 2026-04-06 14:30

### TrendRadar（趋势雷达）

**GitHub**: [sansan0/TrendRadar](https://github.com/sansan0/TrendRadar) | ⭐ 51k+ Stars | 🍴 23k+ Forks

**定位**：AI 驱动的舆情监控与热点追踪工具，告别信息过载，只看真正关心的新闻资讯。

**核心功能**：
- 多平台聚合 + RSS 订阅源
- AI 智能筛选（自然语言描述兴趣，自动提取标签并打分推送）
- AI 分析简报（核心热点态势、舆论风向、异动信号、研判建议）
- 多语言翻译 + Markdown 格式适配
- 推送渠道：微信、飞书、钉钉、Telegram、邮件、ntfy、Bark、Slack、Webhook 等 9+ 种
- MCP 架构支持，可接入 AI 助手进行自然语言对话分析

**技术栈**：
- AI 引擎：LiteLLM（支持 DeepSeek、GPT-4o、Gemini、Claude 等 100+ 提供商）
- 数据来源：基于 newsnow 项目 API

**部署**：Docker（推荐）/ Windows / Mac/Linux / GitHub Codespaces

**许可**：GPL-3.0

**使用场景参考**：可用于舆情监控、竞品追踪、行业趋势分析等场景。
