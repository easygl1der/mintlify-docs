## 2026-04-14 11:15

Onyx is a **comprehensive open-source enterprise AI platform** that combines internal knowledge search, AI chat, deep research, and custom agents — all grounded in your company's own data.

### 核心定位 (Core Purpose)
Onyx 解决了企业内部信息碎片化（Slack, Google Drive, Confluence, GitHub 等）的痛点，通过构建一个**统一的 AI 层**，让员工能够通过自然语言获取带有引用来源的准确答案。

### 底层逻辑：RAG 引擎
Onyx 采用 **检索增强生成 (Retrieval-Augmented Generation)** 机制，而非依赖 LLM 的预训练知识。其检索层包含：
- **混合搜索 (Hybrid Search)**：结合关键词与语义向量搜索。
- **上下文检索 (Contextual Retrieval)**：根据查询意图扩展并重排结果。
- **LLM 生成的知识图谱**：构建组织内部文档、人员与概念的关系网。

### 关键能力矩阵
| 功能 | 核心价值 |
|---|---|
| **企业级搜索** | 跨所有连接数据源的统一搜索 |
| **AI 聊天** | 基于内部文档的 LLM 对话 |
| **Agent 搜索** | 模拟同事的多步代理研究，自动拆解子问题 |
| **自定义 Agent** | 配置特定角色、提示词、数据权限的 AI 助手 |
| **MCP 协议支持** | 允许 Claude Desktop 或 IDE 直接调用 Onyx 知识库 |
| **代码解释器** | 在对话界面直接运行和分析代码 |
| **Web 搜索** | 内部知识与实时 Web 数据的增强融合 |

### 企业级特性与部署
- **数据连接器**：40+ 原生连接器，支持实时同步并**严格同步权限**（确保用户只能看到其在源系统中拥有权限的文档）。
- **模型灵活性**：模型不可知 (Model-agnostic)，支持云端 API (Anthropic, OpenAI, DeepSeek) 或本地部署 (Ollama, vLLM)，满足极高数据隐私要求（如国防、医疗）。
- **部署模式**：MIT 开源，支持 Docker 快速自托管或使用 SaaS 版本。
