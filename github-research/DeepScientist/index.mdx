# DeepScientist V1.5

> 来源：[Datawhale](https://mp.weixin.qq.com/s/oNpMBxBbCePnSpqPZFi7SQ) · 西湖大学张岳实验室 · 2026-04-03

西湖大学张岳团队开源的**自动化科研AI系统**，声称能在7天内完成博士生一年的工作量。

## 核心功能

| 功能 | 说明 |
|------|------|
| 论文复现 | 输入arXiv/GitHub链接，自动复现论文 |
| 代码调试 | 自动跑实验、调试、优化 |
| 数据分析 | 智能分析实验结果，给出优化建议 |
| 论文撰写 | 全自动LaTeX论文生成 |
