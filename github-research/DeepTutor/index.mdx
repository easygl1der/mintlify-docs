# DeepTutor

DeepTutor 是一个开源的智能辅导平台，支持多种 LLM 提供商、本地 Embedding、知识库管理和 AI Agent。

## 核心功能

- **多模型支持** - OpenAI、Anthropic、Ollama 等
- **本地 Embedding** - 支持 Ollama 本地运行
- **知识库 RAG** - 文档上传和检索增强
- **TutorBot Agents** - 可定制的 AI 辅导 Agent
- **多主题** - Light、Dark、Glass、Snow
