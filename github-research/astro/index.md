## 项目简介

**mintlify/astro** 是一个用于将 Mintlify 文档系统集成到 Astro 项目中的官方适配器插件。该项目作为 Mintlify 产品生态的重要组成部分，为 Astro 框架提供了原生的文档集成能力，使开发者能够轻松地在 Astro 项目中使用 Mintlify 强大的文档功能。

### 核心功能

- 提供 `useMintlify()` Astro 集成函数，用于快速接入 Mintlify
- 自动收集 Astro 项目中的路由信息
- 将 Mintlify 文档系统与 Astro 框架进行桥接
- 支持 SSR（服务端渲染）模式
- 提供完整的 TypeScript 类型定义

### 典型使用方式

```javascript
import { defineConfig } from 'astro/config';
import mintlify from '@mintlify/astro';

export default defineConfig({
  integrations: [mintlify()]
});
```
